# Capítulo de Ejemplo

Este es un archivo de ejemplo para demostrar el sistema de preview de borradores.

## Cómo usar este sistema

Coloca tus borradores en formato **Markdown** (.md) en esta carpeta. El sistema los convertirá automáticamente a HTML para revisión.

### Formato de nombres

Se recomienda nombrar los archivos así:

- `01-titulo-capitulo.md`
- `02-otro-capitulo.md`
- `03-tercer-capitulo.md`

El número al inicio determina el orden de aparición.

---

## Elementos soportados

El conversor de Markdown soporta:

- **Texto en negrita**
- *Texto en cursiva*
- ***Negrita y cursiva***

> Las citas se muestran con un borde lateral dorado y estilo especial.
